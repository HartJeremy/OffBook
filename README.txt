Line Rehearsal Trainer PWA v35

How to use:
1. Put this folder on a local web server or hosting service.
   - PWA install/offline features require http://localhost or https://.
   - Opening index.html directly as a file will run the trainer, but install/offline service worker features may not work.
2. Open index.html in Chrome, Edge, or Safari.
3. Use the Install button if your browser shows it.

Quick local test:
- From this folder, run: python3 -m http.server 8000
- Open: http://localhost:8000

Includes current v35 features:
- Actor line numbering
- Rehearsal Navigator
- Section Lookup
- Random, Build-Up Drill, Smart Review
- Pacing run-through
- Speaking/Typing timing boost
- Memory dots saved in browser storage
